# Mission Control — Windows desktop app (download)

## ⬇️ Download the latest build from Releases

**https://github.com/DmitryGofman/mission-control/releases/latest**

Download **`MissionControl-Windows-x64.zip`**, then:

1. **Extract the whole folder** somewhere permanent (Desktop is fine).
2. Double-click **`Mission Control.exe`**.
3. First launch may show Windows SmartScreen → **More info → Run anyway** (unsigned app, one-time).
4. Your data: `mission-control.db` + an `attachments` folder, both next to the app — copy the whole folder to back up.

> Builds are now produced automatically on a Windows machine (GitHub Actions + electron-builder),
> so packaging and the app icon are correct. The old in-branch zip was removed in favour of Releases.
